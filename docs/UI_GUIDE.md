# UI Guide

## Стиль

Elegant Celebration: современно, дорого, минималистично, тепло, празднично без свадебного китча.

## Цвета

- Основной фон: `#F8F6F2`
- Карточки: `#FFFFFF`
- Основной текст: `#1E1E1E`
- Вторичный текст: `#6F6B64`
- CTA: `#4A6CF7`
- Premium accent: `#D6B36A`
- Live screen background: `#121212`

## Шрифты

- Inter для UI.
- Playfair Display только для крупных декоративных заголовков.

## Правила

- Mobile-first.
- Много воздуха и короткие понятные тексты.
- Карточки использовать для сущностей и инструментов, не вкладывать карточки в карточки.
- CTA должен быть явным и не конкурировать с второстепенными действиями.
- Вертикальные фото показывать без поломки композиции.

## Live Screen

- Фон `#121212`.
- Крупный QR.
- Минимум текста.
- Мягкий glow через золотой акцент.
- Фото показывать `object-contain`, background blur можно использовать как атмосферный слой.
